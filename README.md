# DarkTools - Discord Tool Suite

## Features
- Token Checker, Info, Type, Bulk Check from file
- Nitro Link Checker (real check), Generator, Bulk Check
- User Info Lookup, Discriminator Availability Search, Random Username Generator
- Webhook Manager: Send, Embed, Delete, Info, Modify, Spam
- All results saved in `outputs/` folder
